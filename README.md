# 实训表现数字化分析平台

> 为《服务器技术应用》项目一提供数据可视化支持，自动将 8 个测试点归类为 5 大核心技术维度，并以雷达图仪表盘形式展示每位学生的技术薄弱项。

## 功能特性

- **智能数据导入**：支持拖拽或点击上传 Excel 成绩表，自动识别表头并提取第 1–8 题得分。
- **多维度雷达图**：响应式栅格布局，每行展示 3–4 个学生的雷达图，5 层等高线 + 半透明主题色填充。
- **阶梯式自动评价**：
  - `>= 90` 分：熟练掌握（绿色）
  - `>= 60` 分：基本达标（蓝色）
  - `< 60` 分：预警建议（红色），并附带维度针对性改进建议。
- **实时搜索过滤**：顶部搜索框可按姓名或学号快速锁定特定学生。
- **悬停详情浮窗**：鼠标悬停在雷达图顶点时，显示该维度的原始得分与百分制标准化得分。

## 技术栈

- 前端：原生 HTML5 / CSS3 / JavaScript (ES6+)
- 图表库：[ECharts.js](https://echarts.apache.org/)
- Excel 解析：[SheetJS (XLSX.js)](https://sheetjs.com/)
- 部署方式：纯静态页面，无需后端，支持离线运行

## 快速开始

### 方式一：直接打开

用浏览器打开 `index.html` 即可（需联网加载 ECharts 和 SheetJS CDN）。

### 方式二：本地服务器

```bash
python -m http.server 8080
```

然后访问 http://localhost:8080

## 使用说明

1. **加载预设数据**：点击「加载预设数据」按钮，一键展示已解析的 93 名学生（信安 2501 + 2502 班）成绩。
2. **上传自定义 Excel**：点击「上传 Excel」或将 `.xlsx/.xls` 文件拖拽到页面虚线区域。系统会自动从第 4 行开始读取学号、姓名、班级、总分及第 1–8 题得分。

## 数据模型

系统将原始 8 道题目按以下规则映射为 5 个维度（均已百分制标准化）：

| 维度 | 关联题目 | 权重 | 考核技术点 |
|------|---------|------|-----------|
| 磁盘存储管理 | (1)、(2) | 20 分 | `parted` 分区、`/etc/fstab` 挂载 |
| 基础环境构建 | (3)、(7) | 30 分 | YUM 本地源、IP 地址网络配置 |
| 安全合规配置 | (4)、(8) | 20 分 | `firewalld` 防火墙策略 (DNS/HTTP) |
| DNS 服务治理 | (5) | 10 分 | `bind` 域名解析记录与校验 |
| Web 业务应用 | (6) | 20 分 | Apache 站点部署与连通性测试 |
